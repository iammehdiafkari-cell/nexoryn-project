# سامانه اتوماسیون هوشمند صدور و استعلام مدارک - واحد یادگار امام (ره)

این سامانه یک پلتفرم جامع تحت وب جهت صدور، مدیریت و استعلام گواهی‌های اشتغال به تحصیل است که توسط گروه توسعه نرم‌افزاری **نکسورین (NEXORYN)** طراحی و پیاده‌سازی شده است.

## معماری سیستم و امنیت
معماری این سامانه بر پایه Node.js و Express.js بنا شده و از پایگاه داده SQLite جهت مدیریت اطلاعات بهره می‌برد. رویکردهای امنیتی زیر جهت جلوگیری از جعل مدارک پیاده‌سازی شده است:

1. **امضای دیجیتال نامتقارن (RSA-256):** تمامی گواهی‌ها با استفاده از الگوریتم RSA و کلید خصوصی سرور امضا می‌شوند.
2. **هش محتوا (Tamper-evident):** یک هش SHA-256 از داده‌های حیاتی گواهی تولید شده و درون توکن امنیتی (QR Code) قرار می‌گیرد. کوچکترین تغییر در اطلاعات فیزیکی یا دیجیتال گواهی، در سیستم استعلام به عنوان «محتوای مخدوش» شناسایی می‌شود.
3. **کنترل ترافیک (Rate Limiting):** تمامی درگاه‌های API در برابر حملات DOS و Brute-force محافظت شده‌اند.
4. **ثبت وقایع (Audit Log):** تمامی فعالیت‌های سیستم (صدور و استعلام) به همراه آدرس IP در پایگاه داده ثبت و در داشبورد نظارتی قابل پیگیری است.

## پیش‌نیازها و اجرای سیستم
جهت اجرای سامانه نیازی به نصب پیش‌نیازهای متعدد نیست. سیستم با استفاده از Docker کاملاً کپسوله شده است.

### اجرای سریع (از طریق Docker)
1. ترمینال را در مسیر پروژه باز کنید.
2. دستور زیر را وارد نمایید:
   `docker-compose up -d --build`
3. سامانه در آدرس `http://localhost:3000` در دسترس خواهد بود.

### اجرای محلی (بدون Docker)
1. نصب پیش‌نیازها: `npm install`
2. اجرای سرور: `node server.js`
